MoneySur Backend
## Project Setup Check

If the project is not recognized as a Maven project, right-click on pom.xml and select Add as Maven Project.

## Development Server

Navigate to the backend application folder:

cd backend/app


## Build and run the Docker image:

docker compose up


## Run the Spring Boot application:

mvn spring-boot:run
 
## API Testing (Swagger)

To test the project and view all available APIs, open Swagger UI in your browser:

http://localhost:8080/swagger-ui/index.html
